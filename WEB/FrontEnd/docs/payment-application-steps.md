# 支付接口申请详细步骤

## 前期准备

### 1. 企业资质材料（必须）
- **营业执照**：清晰的营业执照照片或扫描件
- **法人身份证**：正反面照片
- **对公账户**：企业银行账户信息
- **门头照片**：店铺或公司门头照片（部分需要）

### 2. 网站信息准备
- **网站域名**：已完成备案的域名
- **网站备案号**：工信部备案号
- **网站截图**：网站首页和主要页面截图
- **商品/服务说明**：详细的业务描述

## 微信支付申请步骤

### 第一步：注册微信支付商户
1. **访问官网**
   - 打开 https://pay.weixin.qq.com/
   - 点击右上角"立即接入"

2. **选择主体类型**
   - **企业**：需要有营业执照
   - **个体工商户**：有个体工商户营业执照
   - **政府/事业单位**：需要组织机构代码证
   - **其他组织**：需要相关证明文件

3. **填写基本信息**
   - **商户全称**：与营业执照完全一致
   - **营业执照注册号**：统一社会信用代码
   - **注册地址**：营业执照上的地址
   - **联系人信息**：法人或负责人信息

### 第二步：填写资质材料
1. **上传营业执照**
   - 格式：jpg、png、jpeg
   - 大小：不超过2MB
   - 要求：四角完整，文字清晰，无反光遮挡

2. **上传法人身份证**
   - 正反面分别上传
   - 格式要求同营业执照
   - 需要看到完整身份证信息

3. **填写结算账户**
   - **对公账户**（推荐）
     - 开户行：完整的银行名称
     - 银行账号：企业对公银行账号
     - 开户名：企业全称
   - **法人个人账户**（部分可用）
     - 需要法人银行卡信息

### 第三步：选择支付产品
1. **扫码支付**（必须）
   - **Native支付**：电脑网站扫码支付
   - 适合：PC端网站支付
   - 费率：0.6%

2. **移动支付**（可选）
   - **H5支付**：手机浏览器支付
   - **JSAPI支付**：微信内支付
   - 适合：移动端网站和微信小程序

### 第四步：网站信息填写
1. **网站基本信息**
   - **网站名称**：与备案一致
   - **网站域名**：已备案的域名
   - **网站备案号**：工信部备案许可证号
   - **网站首页截图**：需要显示域名

2. **业务信息**
   - **经营类目**：选择合适的行业分类
   - **商品名称**：主要销售商品或服务
   - **客服电话**：客户服务联系方式

### 第五步：审核和签约
1. **审核时间**
   - **资质审核**：1-3个工作日
   - **网站审核**：1-5个工作日
   - **总计**：一般2-7个工作日

2. **审核结果**
   - **通过**：可以进入签约环节
   - **驳回**：根据驳回原因修改材料
   - **补充材料**：按要求提供额外材料

3. **签约流程**
   - 下载《微信支付合作协议》
   - 法人签字并加盖公章
   - 上传签约协议
   - 等待签约审核（1-2个工作日）

### 第六步：技术对接
1. **获取商户信息**
   - **商户号**：mch_id（商户唯一标识）
   - **APPID**：公众号或小程序ID
   - **API密钥**：32位字符串，用于签名

2. **下载证书**
   - **API证书**：用于API调用
   - 包含：apiclient_cert.pem、apiclient_key.pem
   - 保存到安全位置，妥善保管

3. **配置回调地址**
   - **支付结果通知URL**：接收支付结果
   - **退款结果通知URL**：接收退款结果
   - 必须是HTTPS地址

## 支付宝申请步骤

### 第一步：注册开放平台账号
1. **访问官网**
   - 打开 https://open.alipay.com/
   - 点击"立即入驻"

2. **选择账号类型**
   - **企业账号**：企业开发者（推荐）
   - **个人账号**：个人开发者（功能有限）

3. **完善账号信息**
   - **企业名称**：与营业执照一致
   - **统一社会信用代码**：营业执照注册号
   - **法人信息**：姓名、身份证号
   - **联系方式**：手机号、邮箱

### 第二步：实名认证
1. **企业认证**
   - 上传营业执照
   - 上传法人身份证
   - 填写对公账户信息
   - 银行小额打款验证

2. **认证时间**
   - **企业认证**：1-3个工作日
   - **打款验证**：1-2个工作日

### 第三步：创建应用
1. **新建应用**
   - 登录开放平台
   - 点击"应用管理" → "创建应用"
   - 选择"网页&移动应用"

2. **填写应用信息**
   - **应用名称**：ZyroFrp内网穿透
   - **应用图标**：应用logo
   - **应用简介**：详细描述应用功能
   - **所属行业**：软件和信息技术服务

3. **上传资质材料**
   - **营业执照**：最新版营业执照
   - **软件著作权**（如有）
   - **ICP备案截图**：网站备案信息

### 第四步：申请支付产品
1. **添加支付产品**
   - **当面付**：扫码支付（必须）
   - **手机网站支付**：移动端支付
   - **电脑网站支付**：PC端支付

2. **填写签约信息**
   - **店铺名称**：与营业执照一致
   - **店铺logo**：店铺标识
   - **店铺简介**：业务描述
   - **客服电话**：联系电话

### 第五步：技术配置
1. **设置密钥**
   - **生成RSA密钥对**：使用支付宝工具生成
   - **上传公钥**：在开放平台上传应用公钥
   - **获取支付宝公钥**：下载支付宝的公钥证书

2. **配置网关地址**
   - **授权回调地址**：用户授权后跳转地址
   - **网关地址**：应用网关地址
   - **异步通知地址**：支付结果通知地址

3. **应用上线**
   - 完成开发后在沙箱环境测试
   - 提交应用审核
   - 审核通过后上线应用

### 第六步：获取开发参数
1. **应用信息**
   - **APPID**：应用唯一标识
   - **网关地址**：https://openapi.alipay.com/gateway.do
   - **应用私钥**：生成的RSA私钥
   - **支付宝公钥**：下载的支付宝公钥

2. **下载证书**
   - **应用证书**：appCertPublicKey.crt
   - **支付宝根证书**：alipayRootCert.crt
   - **证书链**：中间证书

## 申请常见问题

### 1. 申请被驳回怎么办？
- **材料不清晰**：重新拍摄清晰照片
- **信息不一致**：检查所有信息是否与证件一致
- **网站问题**：确保网站已备案且内容合规
- **业务不符**：调整业务描述或选择合适的类目

### 2. 申请时间太长？
- **微信支付**：正常2-7个工作日
- **支付宝**：正常3-10个工作日
- **第三方支付**：2-3个工作日

### 3. 个人可以申请吗？
- **微信支付**：个人不支持，必须企业资质
- **支付宝**：个人账号功能有限，推荐企业账号
- **第三方支付**：部分支持个体工商户

### 4. 费率怎么谈？
- **官方支付**：费率固定，一般不可议价
- **第三方支付**：根据交易量可申请优惠
- **大客户**：交易量大可申请特殊费率

### 5. 需要技术支持怎么办？
- **官方文档**：微信和支付宝都有详细开发文档
- **技术客服**：申请后可联系技术支持
- **第三方服务商**：提供完整技术支持

## 推荐申请顺序

### 新手推荐（最快方案）
1. **易宝支付** → 2-3天开通
2. **技术对接** → 1天完成
3. **上线测试** → 1天验证
4. **总计**：1周内完成

### 企业推荐（长期方案）
1. **微信支付** → 同时申请
2. **支付宝** → 同时申请
3. **技术准备** → 申请期间开发
4. **上线测试** → 审核通过后
5. **总计**：2-3周完成

### 注意事项
- **提前准备材料**：避免因材料不全耽误时间
- **网站备案**：确保网站已完成ICP备案
- **技术准备**：申请期间可以先进行技术准备
- **客服沟通**：遇到问题及时联系客服

选择合适的方案后，按照步骤操作即可成功获得支付接口！
