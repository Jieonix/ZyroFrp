## PayJS 详细接入指南

### 1. 注册和认证
```bash
# 访问官网
https://payjs.cn/

# 注册步骤
1. 点击注册
2. 填写手机号
3. 设置密码
4. 实名认证（上传身份证正反面）
5. 绑定银行卡
6. 等待审核（1-2天）
```

### 2. 获取API参数
```javascript
// 登录后获取以下信息
const payjsConfig = {
  mchid: 'your_mchid',           // 商户号
  key: 'your_api_key',           // API密钥
  notify_url: 'your_notify_url'  // 回调地址
}
```

### 3. 技术对接代码
```javascript
// 创建订单
async function createPayJSOrder(orderData) {
  const params = {
    mchid: payjsConfig.mchid,
    total_fee: orderData.amount,     // 金额（分）
    out_trade_no: orderData.orderNo, // 订单号
    body: orderData.subject,         // 商品描述
    notify_url: payjsConfig.notify_url,
    type: 'native'                   // 扫码支付
  }

  // 生成签名
  const sign = generateSign(params, payjsConfig.key)
  params.sign = sign

  // 发送请求
  const response = await fetch('https://api.mch.weixin.qq.com/pay', {
    method: 'POST',
    body: JSON.stringify(params)
  })

  return response.json()
}

// 查询订单
async function queryPayJSOrder(orderNo) {
  const params = {
    mchid: payjsConfig.mchid,
    out_trade_no: orderNo
  }

  const sign = generateSign(params, payjsConfig.key)
  params.sign = sign

  const response = await fetch('https://api.mch.weixin.qq.com/query', {
    method: 'POST',
    body: JSON.stringify(params)
  })

  return response.json()
}
```
